Attract-Mode Frontend

Attract-Mode is a graphical frontend for command line emulators such as MAME, MESS and Nestopia. It is designed to be run in an arcade cabinet setup and controlled with a joystick or spin dial. Attract-Mode is open source and runs on Linux, Mac OS-X and Windows.

About

Attract-Mode is written in C++ and uses SFML 2.x for graphics/audio and the FFMPEG libraries (optionally) for video support. It is known to compile and run on Ubuntu Linux (32 and 64-bit), Mac OS-X 10.6, and Windows 7 (MinGW).

Features

  • Easy Configuration: most settings can be changed directly from a built-in configuration menu.
  • Tailor the Look and Feel: graphics, sounds, and inputs (joystick, keyboard, mouse) are fully customizable.
  • Features for Cabinet-like Setups: includes screen rotation toggles and an auto-rotation setting to match the rotation of the last game played.
  • Customize to your Collection: display videos, artwork (marquees, screenshots, etc.) and information related to your games.
  • Save Setup Time: automatically generate game lists from directory contents, and import game info from MAME -xmlinfo and MESS -listsoftware commands and from catver.ini files.
  • Built-in Screensaver: uses your artworks and videos.
