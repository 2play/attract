Attract-Mode

Attract-Mode is a graphical front-end for command line emulators such as MAME, MESS and Nestopia. It hides the underlying operating system and is intended to be controlled with a joystick, gamepad or spin dial, making it ideal for use in arcade cabinet setups. Attract-Mode is open source and runs on Linux, OS X and Windows.

Features

  • Configuration menu for easy configuration of most settings.
  • Fully scriptable display layouts written in the Squirrel programming language.
  • Includes the "Attrac-Man" layout, a playable Pacman-inspired game that unfolds around the edges of your game menu.
  • Frontend controls can be mapped to any joystick, keyboard or mouse input.
  • User configurable sounds and ambient music.
  • Display images, videos and information related to specific games.
  • Supports screen rotation and flipping (including auto-rotation).
  • Filter game lists based on multiple game attributes.
  • Plug-ins for integration with external programs (such as LED control or joystick mapping software).
  • Automatically generate game lists from directory contents.
  • Import game information from MAME/MESS -listxml and -listsoftware commands.
  • Catver.ini and nplayers.ini file support.
  • Unicode character (UTF-8) support.
  • Configurable screensaver.

Latest Release

January 17, 2014: The lastest release is version 1.1.1

What's new in 1.1.1:

  • [Windows] Speed up image loading where source directory contains many files.
What's new in 1.1:

  • list filters have been completely reworked. You can now have multiple rules per filter and multiple filters per list. The filters associated with a list can be cycled through and selected in the same manner as lists are cycled/selected.
  • plug-in support added to allow integration with external programs. UltraStik mapping, LEDBlinky and eSpeak plugins are included.
  • much faster MAME romlist creation.
  • fixed behaviour when holding down keys to navigate the game list.
  • fixed movie audio.
  • hide menu bar on OS X.
  • many other bug fixes and improvements.
