Attract-Mode

Attract-Mode is a graphical front-end for command line emulators such as MAME, MESS and Nestopia. It is designed to be run in an arcade cabinet setup and controlled with a joystick or spin dial. Attract-Mode runs on Linux, Mac OS-X and Windows.

Download this project as a .zip file Download this project as a tar.gz file

Features:

  • Fully customizable display, sounds, and input (joystick, keyboard, mouse).
  • Display videos, artwork (marquees, screenshots, etc.) and information related to specific game.
  • Supports screen rotation (including auto-rotation to match the rotation of the last game played). Layouts can further rotate, scale,shear and position images, videos and text.
  • Can automatically generate game lists from directory contents, and import game info from MAME -xmlinfo and MESS -listsoftware commands as well as from catver.ini files.
  • Supports UI localization and unicode characters (UTF-8).
  • Configuration menu for easy configuration of most settings.
  • Configurable screensaver mode that uses random game artworks and videos.
